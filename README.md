# YelpCamp

A full-stack application where users can share and post their camping experiences.

Install dependencies / run node app.js

## Features
  - Authentication
  - Create campground post (general information/image)
  - Signed in user can add comments and rate campgrounds
  - Owner of post or comment can edit/delete entry
  - Overall campground rating
  - Map location using google maps.

## Tools
  - Node/Express
  - EJS (Embedded JS - Template)
  - MongoDB/Mongoose
  - Passport-local-mongoose
  - Connect-flash (user alert notification success/fail)
  - Moment (date reference)
  - BootStrap
  - Starability.css
  - Google Maps iframe
